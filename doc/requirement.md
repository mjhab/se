# 클라우드 파일 공유 시스템 (Mini Drive)

## 1. 기능 요구사항

### 파일 관리

사용자는 다양한 형태의 파일을 업로드하고 저장할 수 있어야 한다.
또한 저장된 파일은 언제든지 다시 다운로드할 수 있어야 한다.

파일이 저장될 때는 다음과 같은 정보가 함께 관리된다.

* 파일 이름
* 업로드 시간
* 파일 크기
* 업로드한 사용자

---

### 폴더 구조

사용자는 파일을 정리하기 위해 폴더를 생성할 수 있어야 한다.
폴더는 여러 단계로 구성될 수 있어야 하며, 파일을 이동시키는 것도 가능해야 한다.

또한 실수로 파일을 삭제하는 경우를 대비하여 일정 기간 동안 복구할 수 있는 기능이 필요하다.

---

### 파일 공유

사용자는 자신의 파일을 다른 사용자와 공유할 수 있어야 한다.

공유 시에는 다음과 같은 권한을 설정할 수 있다.

* 읽기만 가능
* 수정 가능
* 댓글 작성 가능

또한 외부 사용자에게 공유할 수 있도록 링크를 생성하는 기능이 필요하며,
이 링크에는 만료 기간을 설정할 수 있어야 한다.

---

### 검색 기능

파일이 많아질 경우 원하는 파일을 찾기 어려워지므로 검색 기능이 필요하다.

기본적으로 파일 이름으로 검색할 수 있어야 하며,
다음과 같은 조건을 추가로 설정할 수 있다.

* 특정 날짜 이후 업로드된 파일
* 특정 사용자가 업로드한 파일
* 특정 파일 유형

검색 결과는 사용자 경험을 위해 빠르게 제공되어야 한다.

---

### 버전 관리

여러 사용자가 하나의 파일을 수정할 경우 이전 상태를 확인할 수 있어야 한다.

따라서 파일 수정 시 이전 버전을 저장하고,
사용자는 이전 버전을 확인하거나 다운로드, 복원할 수 있어야 한다.

단, 저장 용량을 고려하여 버전은 일정 개수 또는 기간 동안만 유지한다.

---

### 사용자 관리

시스템은 사용자 계정을 기반으로 동작한다.

사용자는 이메일과 비밀번호를 이용해 로그인할 수 있으며,
관리자는 사용자 계정을 생성하거나 삭제할 수 있다.

또한 사용자별 저장 공간을 관리할 수 있어야 하며,
필요 시 사용자 그룹을 구성할 수 있다.

---

## 2. 비기능 요구사항

### 보안

* 인증된 사용자만 시스템에 접근 가능해야 한다.
* 사용자 권한에 따라 접근 가능한 기능이 제한되어야 한다.
* 파일 전송 과정에서 데이터는 암호화되어야 한다.
* 외부 공유 링크는 만료 기간을 통해 접근을 제한한다.

---

### 성능

* 약 200명 정도의 사용자가 동시에 이용해도 안정적으로 동작해야 한다.
* 특히 파일 검색과 다운로드 기능은 빠른 응답 속도를 유지해야 한다.

---

## 3. 기존 서비스 분석

### Google Drive

Google Drive는 권한을 세분화하여 협업에 유리한 구조를 가지고 있으며,
파일 검색 기능이 매우 강력하다는 특징이 있다.
또한 파일 버전을 관리할 수 있어 변경 이력을 확인하기 쉽다.

→ 이를 통해 권한 설정과 검색 기능은 필수 요소라고 판단된다.

---

### Dropbox

Dropbox는 외부 공유 링크를 통해 파일을 전달할 수 있으며,
링크 만료와 같은 보안 기능을 제공한다.
또한 버전 복원 기능이 안정적으로 동작한다.

→ 따라서 외부 공유 시 보안 정책을 명확하게 설정하는 것이 중요하다.

---

### OneDrive

OneDrive는 내부 사용자 공유와 외부 공유를 구분하여 관리할 수 있으며,
권한 설정이 직관적이다.
또한 다양한 파일 형식을 지원하고 안정적인 동기화 기능을 제공한다.

→ 내부 협업과 외부 공유를 구분하는 구조가 필요하다고 판단된다.

---

## 4. 결론

본 시스템은 파일 저장, 관리, 공유, 검색, 보안 기능을 중심으로
조직 내 협업을 효율적으로 지원하는 것을 목표로 한다.

또한 향후 기능 확장을 고려하여 유연한 구조로 설계하는 것이 중요하다.

## 기능 우선순위

프로젝트 구현 시 모든 기능을 한 번에 개발하기보다, 핵심 기능부터 우선적으로 구현하는 것이 효율적이다.
이에 따라 기능 우선순위를 다음과 같이 정리할 수 있다.

### 1순위 (핵심 기능)
- 파일 업로드 및 다운로드
- 폴더 생성 및 관리
- 사용자 간 파일 공유
- 로그인 및 권한 관리
- 파일 이름 및 조건 기반 검색

### 2순위 (중요 기능)
- 파일 버전 관리
- 외부 공유 링크 생성 및 만료 기간 설정
- 삭제 파일 복구 기능

### 3순위 (확장 기능)
- 사용자 그룹 기능
- AI 기반 검색
- 자동 파일 분류 기능
- 문서 공동 편집 기능
