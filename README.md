# SE Assignment Repository

## 프로젝트 개요
클라우드 파일 공유 시스템(미니 드라이브)의 요구사항을 분석하고 문서화하는 프로젝트입니다.

---

## 프로젝트 구조

- `doc/` : 과제 관련 문서 저장
- `src/` : 소스 코드 저장 (현재 미구현)

---

## 과제 결과물 위치

- 요구사항 명세서: `doc/requirement.md`

---

## 빌드 및 실행 방법

현재는 요구사항 분석 단계로, 구현된 소스 코드는 없습니다.  
추후 `src/` 폴더에 코드가 추가되면 빌드 및 실행 방법을 작성할 예정입니다.
